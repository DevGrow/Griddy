Griddy: Fast and Simple Grid Overlay Plugin for JQuery

Griddy is a simple JQuery plugin that overlays a customizable grid over any element. The current options and defaults are: 

$('.element').griddy(
	rows: 10,
	rowheight: 0,
	rowgutter: 20,
	columns: 4,
	columnheight: 0,
	columngutter: 20,
	color: '#ccc', 
	opacity: 30
	}
Use the default 0 for rowheight or columnheight to make Griddy calculate the column width and row height automatically.
